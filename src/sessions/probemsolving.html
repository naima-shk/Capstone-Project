PROBLEM SOLVING

DESCRIPTION


Problem Solving is the process of finding a solution to issues that are critical and needs to be worked on urgently also requires attention. If a particular person has the ability to cope up with any difficult situation, then that person is said to have problem-solving skills. Problem-solving is considered one of the most valuable skills and organizations consider this skill while hiring an employee. These are the steps for problem-solving:

  • Identify the problem
  • The reason behind the occurrence of the problem
  • List all the solutions possible for solving that problem
  • Choose the best fit
  • Implement the solution
  • Evaluation of results
